Graph Convolutional Neural Networks for Recommender Systems

 

In this project the application of graph convolutional neural networks to recommender systems is discussed. T. Kipf and M. Welling introduced graph convolutional neural networks in [1] and [2] and applied this method to semi-supervised classification tasks and link prediction on general graphs. Here, we apply these methods to recommender systems, for which the interaction data can be represented by bipartite graphs.
